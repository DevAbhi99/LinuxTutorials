## If using other operating system other than Linux 

## First of all install docker engine from the official website and start docker

## then go to the attached url:- https://hub.docker.com/r/techworldwithnana/youtube/tags

## open cmd in case of windows or open any terminal and copy the command and execute it:- docker pull techworldwithnana/youtube:linux-tutorial

## then execute the command docker run -it techworldwithnana/youtube:linux-tutorial (linux terminal will be running in the docker container)
